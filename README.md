# LLM Learning Assistant App

## Overview
This Android app was developed for SIT305 Task 6.1D.  
The app is designed to support student learning through personalised content, adaptive practice, and LLM-style study support.

## Features
- Student login screen
- Student registration screen
- Interest selection screen
- Personalised home screen
- AI-generated task screen
- Results screen

## LLM-Powered Learning Utilities
The app includes the following LLM-based learning features:
1. Generate hint for a question
2. Explain why an answer is correct or incorrect
3. Create 3 flashcards from a selected topic

## UI/UX Features
- Prompt and response displayed in the UI
- Loading states shown during AI generation
- Failure states shown when generation fails
- Clear navigation between screens
- Readable text and meaningful buttons

## Technical Details
- Built with Kotlin and XML
- Uses dummy data for demonstration
- Designed for future backend API integration

## Project Structure
- LoginActivity
- RegisterActivity
- InterestsActivity
- HomeActivity
- TaskActivity
- ResultActivity
- DummyData
- Question model

## Notes
This version uses simulated AI responses and dummy data to demonstrate the required learning utilities and user experience.
